## Cringe Flicks

A mini blog site built with Nextjs and sanity.io for content management.

## Technologies
* Typescript
* Tailwind CSS
* Sanity.io CMS

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Navigate to the studio folder and run

```bash
npm run dev
# or
yarn dev
```
